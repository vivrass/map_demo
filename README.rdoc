== HOW TO

1. Install git
2. cd WHERE_YOU_WANT_THE_PROJECT
3. git clone https://github.com/vivrass/map_demo.git (Get the source code)
4. cd map_demo
5. bundle install (Install the gems)
6. bundle exec rake db:create
7. bundle exec rake db:migrate
8. bundle exec rails server (Start the server)
9. http://127.0.0.1:3000


You can also run 'bundle exec rake db:seed' to add 10 entries to your database
